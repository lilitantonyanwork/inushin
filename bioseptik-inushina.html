Биосептик Инюшина

Продукт для гигиены

Новая разработка Инюшина В. М.

  • / Каталог продукции лаборатории Инюшина
  • / Биосептик Инюшина.

"Биосептик Инюшина" - это новая разработка Инюшина В.М., для очищения эко среды от инфицированных различного рода бактерий и других микроорганизмов и вирусов.

Для нейтрализации микроорганизмов, используется принцип анабиоза, то есть с помощью такого рода биофизических гигиенических действий антисептика удается перевести агрессивные бактерии и вирусы, в том числе и коронарный вирус, в состояние анабиоза - в состояние «спячки», которое может продолжаться 10 - 12 лет и более.

"Биосептик Инюшина" полностью безопасен для живого организма и окружающей эко среды, которая тоже может быть заражена различного рода инфекциями или загрязнена различного рода компонентами, типа канцерогенов и т. д.

Для этих целей используется биофизический продукт (комплекс) для гигиены.

Использовать только для наружного применения для обеззараживания поверхности кожи тела, а также поверхности различных предметов (мебель, сотовые телефоны, компьютеры, приборы и т.д.)

Основной состав продукта:

Глицерин, растительные предельные масла, питьевая вода с низким содержанием минералов, несколько разновидностей гидроплазмы.

Объём: 100 мл.

Температура хранения: от 0°С до +30°С.

Срок годности: 3 года.

Меры предосторожности:

Избегайте попадания в глаза, при попадании в глаза тщательно промойте водой.

Только для наружного применения.

Натуральный продукт, не содержит вредных для здоровья горючих и токсичных веществ.

Изготовлено в лаборатории Инюшина В. М., по заказу компании "Источник Здоровья".
